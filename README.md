# saasoftTest
Тестовое задание для SaaSoft
===

## Примечания
===
1. Время на выполнение - приблизительно 9-10 часов.
2. Вся основная логика сконцетрирована в компоненте BlockAccount и в Store, там же есть комментарии по коду.
3. Правильно заполненные данные сохраняются в localStorage и автоматически подгружаются оттуда при открытии страницы.
4. Для UI-компонентов взята библиотека Vuetify, для валидации - Vuelidate. С ними есть большой опыт на других проектах, поэтому выбрал именно их.
5. Очень хотелось бы услышать обратную связь, что можно улучшить, если где сделано плохо или имеются баги - тоже буду очень признателен, если напишете об этом :)
