# saasoftTest
Тестовое задание для SaaSoft
===

## Примечания
1. Время на выполнение - приблизительно 9-10 часов.
2. Таблица с данными сделана с помощью CSS Grid. Изначально планировал сделать через `<v-table><tr><td>`, но внутри таблиц нельзя использовать тег `<form>` для каждой отдельной строки. В целом, можно и переделать на `<table>`, но тогда поля не будут внутри тега `<form>`, а так делать не рекомендуется.
3. Вся основная логика сконцетрирована в компоненте BlockAccount и в Store, там же есть комментарии по коду.
4. Правильно заполненные данные сохраняются в localStorage и автоматически подгружаются оттуда при открытии страницы.
5. Для UI-компонентов взята библиотека Vuetify, для валидации - Vuelidate. С ними есть большой опыт на других проектах, поэтому выбрал именно их.
6. Очень хотелось бы услышать обратную связь, что можно улучшить, если где сделано плохо или имеются баги - тоже буду очень признателен, если напишете об этом :)
